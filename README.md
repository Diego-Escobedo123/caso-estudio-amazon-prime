# Caso de Estudio 1 - Amazon Prime

Proyecto desarrollado para el curso Introducción a los Datos basado en una simulación de un sistema de recompensas inspirado en Amazon Prime.

## Descripción
Se implementó una prueba funcional (POC) utilizando Python y Google Colab para simular:
- Generación de datos de usuarios
- Limpieza y transformación de datos
- Sistema de puntos
- Segmentación de usuarios
- Detección de churn
- Visualización de datos

## Tecnologías Utilizadas
- Python
- Pandas
- NumPy
- Matplotlib
- Google Colab

## Funcionalidades
- Simulación de usuarios y compras
- Cálculo de puntos por actividad
- Clasificación de usuarios
- Identificación de usuarios en riesgo
- Generación de gráficas y análisis
